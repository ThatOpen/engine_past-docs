Past That Open Docs versions

Here you will find the documentation to previous versions of the libraries. Keep in mind the following points:

  • We strongly recommend to use the latest versions of the libraries.
  • Each new version comes with a migration guide from the previous one.
  • We do not make bug fixes for these versions - all fixes go to the newer ones.
  • We do not guarantee that the live examples will work, as they are bound to the latest version of the libraries.
  • These docs don't have a search function because Algolia just works for the updated docs page.

If you have any questions, let us know in our community.
